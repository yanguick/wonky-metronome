# wonky-metronome

A metronome web app built with React, TypeScript, and Vite. Installable as a PWA.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech stack

- React 18
- TypeScript
- Vite
- vite-plugin-pwa
- Web Audio API for sample-accurate click scheduling
